## Summary

Created the initial `frontend/synapse/` module skeleton to begin separating Synapse data, rendering, controls, labels, selection, companion surface, perception panel, and museum preset.

This does not replace the current canvas renderer yet.

The current prototype remains intact.

## Purpose

The canvas renderer proved the system.

The future Three.js renderer must prove the feeling.

v0.4.0 creates the architectural boundary required for that migration.
