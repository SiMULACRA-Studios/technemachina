## Summary

The Synapse Companion Shell now handles larger-brain/provider failures gracefully.

When the external daemon brain is busy, unreachable, or returns provider overload text, the companion no longer dumps raw API errors into the user-facing chat.

Instead, she falls back to local Synapse mode.
